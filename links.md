# OSINT & Cybersecurity Resource List

**Disclaimer:** Many of these tools are extremely powerful. Always use them responsibly, ethically, and in accordance with local and international laws. Some services may be freemium or require a subscription for full access.

## Internet-Wide Scanners & Asset Discovery
*   [Shodan](https://shodan.io/) - The original search engine for internet-connected devices, servers, and the Internet of Things (IoT).
*   [Censys](https://search.censys.io/) - Scans the internet to find devices and networks, providing a global view of the infrastructure.
*   [ZoomEye](https://www.zoomeye.org/) - A cyberspace search engine from China that gathers information about targets and exposed assets.
*   [Onyphe](https://www.onyphe.io/) - A Cyber Defense Search Engine for open-source and cyber threat intelligence data.
*   [Fofa](https://fofa.info/) - A cyberspace search engine with a focus on threat intelligence and vulnerability research.
*   [GreyNoise](https://viz.greynoise.io/) - Analyzes internet-wide scan and attack traffic to differentiate targeted threats from background noise.
*   [Netlas](https://netlas.io/) - A search and monitoring platform for any internet-connected asset, including passive DNS and WHOIS.
*   [IVRE](https://ivre.rocks/) - A network reconnaissance framework with a search engine to query its scan results.
*   [Quake](https://quake.360.net/) - A cyberspace surveying and mapping system from Qihoo 360.
*   [Hunter.how](https://hunter.how/) - Performs fingerprint retrieval of internet-connected devices and services.
*   [ODIN](https://odin.so/) - A powerful search engine for scanned internet assets.
*   [Thingful](https://thingful.net/) - A search engine specifically for the Internet of Things (IoT).
*   [Natlas](https://github.com/natlas/natlas) - An open-source, scalable asset discovery and scanning platform.

## Vulnerability & Exploit Databases
*   [NIST NVD](https://nvd.nist.gov/vuln/search) - The National Vulnerability Database, a comprehensive U.S. government repository of vulnerability data.
*   [CVE Mitre](https://cve.mitre.org/cve/search_cve_list.html) - The official catalog for Common Vulnerabilities and Exposures (CVEs).
*   [Exploit-DB](https://www.exploit-db.com/) - An extensive, searchable archive of public exploits and vulnerable software.
*   [Vulners](https://vulners.com/) - A massive, continuously updated database of vulnerabilities, exploits, and security intelligence.
*   [Sploitus](https://sploitus.com/v) - A search engine that aggregates exploits and vulnerabilities from multiple sources.
*   [Snyk](https://security.snyk.io/) - A developer-focused platform for finding vulnerabilities in open source dependencies and containers.
*   [OpenCVE](https://www.opencve.io/cve) - A platform to track CVE updates and be alerted to new vulnerabilities.
*   [OSV.dev](https://osv.dev/list) - A distributed vulnerability database specifically for open source projects.
*   [GitHub Advisory Database](https://github.com/advisories) - A database of security vulnerabilities, including CVEs and GitHub-originated advisories.
*   [CloudVulnDB](https://www.cloudvulndb.org/) - A database for vulnerabilities and security issues specific to cloud platforms.
*   [Packet Storm Security](https://packetstormsecurity.com/) - A long-standing resource for the latest vulnerabilities, exploits, and security tools.
*   [Trend Micro Zero Day Initiative](https://www.zerodayinitiative.com/advisories/published/) - Publicly disclosed vulnerabilities discovered by ZDI researchers.
*   [Rapid7 DB](https://www.rapid7.com/db/modules/) - The module database for the Metasploit Framework.
*   [CXSecurity](https://cxsecurity.com/exploit/) - A database of exploits and vulnerabilities.
*   [Vulmon](https://vulmon.com/) - A vulnerability search engine with a focus on patch intelligence.

## Threat Intelligence & Malware Analysis
*   [VirusTotal](https://www.virustotal.com/) - Analyzes files and URLs for malware with over 70 antivirus scanners.
*   [AbuseIPDB](https://www.abuseipdb.com/) - A project dedicated to helping combat the spread of hackers, spammers, and abusive activity on the internet.
*   [AlienVault OTX](https://otx.alienvault.com/) - Open Threat Exchange, one of the world's largest open threat intelligence communities.
*   [Pulsedive](https://pulsedive.com/) - A free, community-driven threat intelligence platform for enriching IOCs.
*   [URLScan.io](https://urlscan.io/) - A free service to scan and analyze websites for malicious activity and technologies used.
*   [Any.Run](https://any.run/) - An interactive online malware analysis sandbox.
*   [Joe Sandbox](https://www.joesandbox.com/) - A deep malware analysis sandbox for suspicious files and URLs.
*   [ThreatConnect](https://threatconnect.com/intel-search/) - A search engine for threat intelligence indicators.
*   [ThreatMiner](https://www.threatminer.org/) - A data mining portal for threat intelligence.
*   [PhishTank](https://phishtank.org/) - A collaborative clearing house for data and information about phishing on the Internet.
*   [Maltiverse](https://maltiverse.com/) - An IOC search engine that aggregates data from multiple sources.
*   [Hudson Rock](https://www.hudsonrock.com/) - A cybercrime intelligence company specializing in data from infostealer malware.
*   [SOCRadar](https://socradar.io/) - An extended Threat Intelligence (XTI) platform for attack surface management.
*   [LeakIX](https://leakix.net/) - A search engine that indexes publicly available information from various misconfigured sources.

## People, Social Media & Facial Recognition
*   [PimEyes](https://pimeyes.com/en) - A reverse image search engine that uses facial recognition to find where an image appears online.
*   [Social Searcher](https://www.social-searcher.com/) - A social media search engine for monitoring mentions and hashtags.
*   [Epieos](https://epieos.com/) - Finds social media accounts associated with an email address.
*   [That's Them](https://thatsthem.com/) - A reverse phone number and people search engine.
*   [Pipl](https://pipl.com/) - A professional-grade people search engine for identity verification.
*   [Tinfoleak](https://tinfoleak.com/) - Provides a detailed analysis of a Twitter user's profile and activity.
*   [TweetDeck](https://tweetdeck.twitter.com/) - An advanced Twitter client for managing multiple accounts and searches.
*   [Reddit Investigator](http://reddit-user-analyzer.netlify.app/) - Analyzes a Reddit user's comment history and statistics.
*   [Picuki](https://www.picuki.com/) - An anonymous Instagram story viewer and profile browser.

## Domain, IP, Email & Certificate Search
*   [SecurityTrails](https://securitytrails.com/) - Provides a comprehensive inventory of internet assets, including current and historical DNS and WHOIS data.
*   [DNSDumpster](https://dnsdumpster.com/) - A free tool for quick and comprehensive DNS record searching.
*   [ViewDNS.info](https://viewdns.info/) - A collection of dozens of DNS and IP-related tools.
*   [MXToolbox](https://mxtoolbox.com/) - A comprehensive tool for DNS, email, and network diagnostics.
*   [crt.sh](https://crt.sh/) - A search engine for public SSL/TLS certificates that have been logged by Certificate Transparency logs.
*   [Have I Been Pwned](https://haveibeenpwned.com/) - A service that allows you to check if your email has been compromised in a data breach.
*   [Hunter.io](https://hunter.io/) - Finds professional email addresses associated with a domain.
*   [EmailRep.io](https://emailrep.io/) - An email reputation search engine to detect risky users.
*   [IntelligenceX](https://intelx.io/) - A powerful search engine and data archive that searches Tor, I2P, data leaks, domains, and emails.
*   [PhoneBook.cz](https://phonebook.cz/) - Lists all domains, email addresses, and URLs associated with a given domain.
*   [RobTex](https://www.robtex.com/) - A tool for various kinds of research on IP numbers, domain names, and other internet artifacts.
*   [CentralOps](https://centralops.net/co/) - A collection of network diagnostic and investigation tools.
*   [Whoisology](https://whoisology.com/) - Provides deep connections between domain name, email, and owner data.

## Code Search Engines
*   [GitHub](https://github.com/search) / [GitLab](https://gitlab.com/explore) - The largest platforms for hosting code, offering powerful search capabilities.
*   [Grep.app](https://grep.app/) - Searches across over half a million public Git repositories.
*   [SearchCode](https://searchcode.com/) - A free source code search engine indexing billions of lines of code from open source projects.
*   [PublicWWW](https://publicwww.com/) - Finds websites based on the source code, scripts, and technologies they use.
*   [Sourcegraph](https://sourcegraph.com/search) - A code intelligence platform with powerful code search capabilities.

## Data Leaks & Public Storage
*   [Dehashed](https://www.dehashed.com/) - A search engine for viewing credentials, personal information, and other data from leaked databases.
*   [GrayHatWarfare](https://buckets.grayhatwarfare.com/) - A search engine for publicly accessible AWS S3 buckets and their contents.
*   [BreachDirectory.org](http://breachdirectory.i2p/) - A search engine for data breaches, accessible via Tor.

## Geospatial & Location-Based OSINT
*   [Google Maps](https://maps.google.com/) / [Google Earth](https://www.google.com/earth/) - The standard for street-level and satellite imagery.
*   [Yandex Maps](https://yandex.com/maps) - Offers high-quality satellite and street-level imagery, especially in Eastern Europe.
*   [Sentinel Hub](https://www.sentinel-hub.com/explore/eobrowser/) - Provides access to a wide range of satellite imagery from various sources.
*   [FlightRadar24](https://www.flightradar24.com/) - A global flight tracking service that provides you with real-time information about thousands of aircraft.
*   [MarineTraffic](https://www.marinetraffic.com/) - A service that tracks the position of ships and vessels worldwide.
*   [SunCalc](https://www.suncalc.org/) - A tool that shows sun movement and sunlight phases for a given day at a given location.

## Web History & Archives
*   [Wayback Machine](https://archive.org/web/) - An archive of the internet, allowing you to view content from deleted or changed websites.
*   [Archive.today](https://archive.ph/) - An on-demand webpage archiver that preserves text and a graphical copy of pages.
*   [Google Cache](https://www.google.com/) - Use the `cache:website.com` dork to see Google's last cached version of a site.
*   [UK Web Archive](https://www.webarchive.org.uk/ukwa/) - Collects millions of websites each year, preserving them for future generations.

## File & Metadata Analysis
*   [ExifTool](https://exiftool.org/) - The industry standard for reading, writing, and editing meta information in a wide variety of files.
*   [Jeffrey's Image Metadata Viewer](http://exif.regex.info/exif.cgi) - A web-based tool for viewing metadata in image files.

## Frameworks & Toolkits
*   [OSINT Framework](https://osintframework.com/) - A comprehensive, web-based mind map of OSINT tools and resources.
*   [Maltego](https://www.maltego.com/) - A graphical link analysis tool for visualizing OSINT data.
*   [SpiderFoot](https://www.spiderfoot.net/) - An open-source OSINT automation tool that can be self-hosted or used as a cloud service.
